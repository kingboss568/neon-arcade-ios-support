## Summary

NEON ARCADE 22 Games is a fully offline iOS arcade collection. All 22 games are embedded in the app binary as local HTML, CSS, and JavaScript assets packaged by Capacitor. The app does not stream, download, or execute remote games or software. There are no external mini-game URLs and no external game developers.

Local packaged entry point:

- `www/index.html`
- `www/js/engine.js`
- `www/js/games1.js`
- `www/js/games2.js`
- `www/js/games3.js`
- `www/js/games4.js`
- `www/js/games5.js`
- `www/js/games6.js`

The only network-dependent feature is the App Store in-app purchase flow for the non-consumable product `com.kingboss.neonarcade.unlockall22`; it unlocks the premium games already embedded in the binary.
